# -Exam-Paper-Generator-from-Syllabus
Exam Paper Generator from Syllabus is a web-based application that automatically generates question papers based on a selected syllabus, chapters, marks distribution, difficulty level, and question types. It helps teachers create balanced, customizable exam papers within seconds, reducing manual effort while ensuring syllabus coverage.

Exam Paper Generator from Syllabus

Overview

Exam Paper Generator from Syllabus is a web application that automatically generates examination papers based on a predefined syllabus and question bank. Teachers can select chapters, marks distribution, difficulty levels, and question types to instantly create balanced question papers.

Workflow

1. Teacher logs in.
2. Select subject.
3. Select chapters.
4. Choose total marks.
5. Select difficulty level.
6. Click Generate Paper.
7. System randomly selects questions.
8. Preview paper.
9. Export to PDF or Print.
